Canvas

This project contains some work I did to learn how to use HTML canvas. These are short one-off projects so I put them all together in one repo.

  • Clock

    An old-fashoined clock with hands

  • Canvas

    A simple sketching program. It includes color and line width selection, allows loading of a background image and has basic undo functionality.

  • Canvas(mobile)

    Similar to canvas, but for touchscreen mobile devices.

These are all served from Github Pages. The repository is at github.com/davidthaler/canvas.
