Canvas

This project contains some work I did to learn how to use HTML canvas. These are short one-off projects so I put them all together in one repo.

  • Clock

    An old-fashoined clock with hands

  • Canvas

    (Almost) the simplest sketching program you can make. It does include color and line width selection.

  • Canvas(mobile)

    Similar to canvas, but for touchscreen mobile devices.

Thes are all served from Github Pages. The repository is at github.com/davidthaler/canvas.
